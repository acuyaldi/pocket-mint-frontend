## Known deviations

- Di lingkungan lokal (Windows), `embedded-postgres` yang dipakai
  `run-integration-tests.mjs` mengalami hang pada tahap teardown
  (`pg.stop()`) setelah test selesai â€” bukan pada tahap migrasi atau
  eksekusi test (hasil test sudah tercatat lengkap sebelum hang terjadi).
  Proses dihentikan manual dan data Postgres sementara dibersihkan. Ini
  quirk lingkungan lokal, bukan masalah repository: CI memakai service
  container Postgres asli yang dibongkar oleh GitHub Actions sendiri, jalur
  yang sudah diverifikasi terpisah lewat isi `ci.yml` dan sebelumnya lewat
  smoke test nyata di `mvp-stable-rc-validation.md`.
- `npm run build` di clone backend menghasilkan diff pada `dist/` dan
  `src/generated/prisma/*` dibanding versi yang ter-commit (embedded
  base64/WASM Prisma client tidak byte-identik antar run generate). Ini
  terjadi di clone disposable dan tidak diaplikasikan balik ke repo asli;
  dicatat sebagai known non-determinism pada output generated Prisma
  client, bukan indikasi source code yang belum di-build.
- Clone dilakukan dari working copy lokal (`git clone` ke path lain), bukan
  dari `origin` di GitHub â€” memverifikasi reproducibility dari riwayat Git
  yang ter-push secara lokal setara dengan commit `HEAD` di atas, tanpa
  memerlukan akses jaringan/kredensial GitHub tambahan pada sesi ini.
